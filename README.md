# Turkey's Flight Automation
## Overview
A flight automation mod that can automate various tasks such as ascent guidance, orbital maneuvers, and rendezvous with high levels of user customizability and reliability.
## Features
- currently none
## Upcoming features
- various info displays with relevant information
- hohmann transfer automation

* this project was originally just going to be a port of the ksp1 mod MechJeb, but it differs enough that that is no longer the case. there are currently still many artifacts of this
